# 1
The node js master class 

## This is the first assingment of The Node.js Master Class

## Instructions

* Pull the repo
* Cd into the project folder
* Run the following command

_node index.js_


## Expected behavoiur 

When posting to localhost:3000/hello

You will recive a json message like this

_"message": "Hello user :) "_

If you try another method that is NOT post, you will recive this message

_"message": "This api only accepts post requests"_



And if you try a non existing url, you will recive this message

_"message": "Route not found "_
